# About the Project
## What this project is
### Single-page website "Hollow Knight: Path of Pain" — a fandom page with a detailed guide for completing the Path of Pain.
> [!NOTE]
> ## Purpose
> - Study all stages of the location through annotated images and a video guide.

> [!TIP]
> ## What is included (planned features)
> - Header menu:
> 	- positioned not at the very top of the page, but above the main content block (body)
>	- site logo
>	- dropdown lists with categories and pages
> - Main content block (body):
>	- Information section:
>	  - description of what the "Path of Pain" is and an "Additional" field
>	  - description of where to find the location
>	  - description of rewards for completion
>	  - which Charms are useful for completion
>	  - step-by-step walkthrough with illustrations
>	  - video guide following the text version
>	  - image gallery with scrollable navigation
>	  - "Hallownest locations"
>	- "Additional block":
>	  - Related topics
>	  - You might also like
>	  - Popular topics
>	- Additional:
>	  - location image
>	  - location soundtrack
>	  - table of contents
>	  - page language switcher
>	  - "Fix" button
>	- Footer:
>	  - Contact us
>	  - Useful links
> - Additional requirements:
>	- custom scrollbar in purple shades
>	- the content block background should be semi-transparent
>	- the page background should feature a frame from the "Path of Pain"

> [!IMPORTANT]
> When creating this website, I was inspired by the official fandom site [Hollow Knight Wiki | Fandom](https://hollowknight.fandom.com/wiki/Hollow_Knight_Wiki). Some images, all links, and the design are taken directly from it. Also, for the "Step-by-step game walkthrough" section, I used an excellent guide from the Steam Community: [Прохождение hollow knight на 112% (+DLC) (с картинками)](https://steamcommunity.com/sharedfiles/filedetails/?id=2164362413).
