# О проекте
## Что из себя представляет проект
### Одностраничный сайт «Hollow Knight: Путь боли» - фандом-страница с подробным гайдом по прохождению пути боли. 
> [!NOTE]
> ## Для чего нужен
> - Изучение всех этапов локации по изображениям с указателями на них и по видео-гайду.

> [!TIP]
> ## Что имеется (должно быть)
> - Заголовочное меню:
> 	- не в самом верху страницы, а над основным информационным блоком (body)
>	- логотип сайта
>	- выпадающие списки с категориями и страницами
> - Основной информационный блок (body):
>	- Зона с информацией:
>	  - описание того, что такое "Путь боли" и поле "Дополнительно"
>	  - описание того, где найти локацию
>	  - описание того, что дают за прохождение
>	  - какие амулеты пригодятся для прохождения
>	  - поэтапный рассказ прохождения с иллюстрациями
>	  - видео-гайд после текстового варианта
>	  - галлерея с пролистыванием изображений
>	  - "локации Халлоунеста"
>	- "Доп. блок":
>	  - Связанные темы
>	  - Вам может понравиться
>	  - Популярные темы
>	- Дополнительно:
>	  - изображение локации
>	  - саундтрек локации
>	  - содержание
>	  - смена языка страницы
>	  - кнопка "Исправить"
>	- Подвал:
>	  - Связь с нами
>	  - Полезные ссылки
> - Дополнительные требования:
>	- scroll должен быть кастомным в фиолетовых оттенках
>	- информационный блок, а именно фон, должен быть полу-прозрачным
>	- фон страницы должен содержать кадр из "Пути боли"
